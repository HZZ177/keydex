Agent全链路运行时异常告警与告警配置管理

何何守一 · 负责人 进行中 P0 紧急 Sprint07_0.7.x

背景

目前 Agent 在运行过程中出现异常时，缺少统一的告警上报、通知与配置管理能力，导致异常无法及时被发现和处理，影响系统稳定性和用户体验。

验收标准

  • 对话链路异常实时捕获并生成告警，覆盖模型欠费、限流、API 错误、MCP 工具下线、沙盒失败等场景。
  • 告警支持 P0/P1/P2 分级，支持重复去重、恢复闭环。
  • 支持企业微信、邮件、飞书、Webhook 等多渠道通知配置。
  • 支持告警规则配置与路由策略维护。
  • 提供告警历史查询与详情查看。

范围

包含对话链路、异步任务链路与核心后端服务，不包含第三方平台自身故障修复。
