项目模式产品方向说明

Keydex 项目模式是连接云端项目管理平台和本地 AI 执行环境的个人工作台。

核心结论

  • 云端管理项目事实，Keydex 管理本地执行闭环。
  • Agent 在项目模式中保持底部胶囊形态，并感知当前页面。
  • 长程任务应从云端对象无感启动，缺信息时才轻量确认。

Keydex 长程任务

用户管理的是长程任务；Session 是背后的对话和执行记录，不直接污染普通会话列表。
