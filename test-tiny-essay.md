# 窗台上的五分钟

每天早晨，我习惯在窗台上站五分钟。不是打卡，不是修行，就是站在那里发呆，什么也不想。

窗台很窄，刚好搁下一只马克杯。热茶的蒸汽在玻璃上晕出一小片雾，模糊了对面的天台。老屋顶的沥青补丁有新有旧，像打了许多补丁的旧衣裳。三楼晾着一条花裙子，四楼的老爷爷在给鸽子撒玉米。

这些场景每天都在重复，像一盘反复播放的磁带。可怪就怪在，我从来不觉得烦。也许是它们太稳定了，反而成了定心丸。这座城市里，什么东西都变得快——外卖涨价、巷口的煎饼店关了、收藏夹里的歌下架了。反倒是这排旧楼，像一头沉默的老牛，卧在城市的角落里，告诉你：不急。

有时候我在想，人为什么需要一扇窗。不是为了看远方——远方太远，看不清。而是为了知道今天和昨天不一样。今天的云层比昨天薄，今天的风比昨天软，今天的空气里隐隐约约有一丝炒栗子的焦香。

窗外那棵老梧桐落光了叶子，枝干像一幅速写的炭笔线条。等到春天，它会重新冒出新芽，把整条街罩进一片嫩绿里。那时候的早晨，会有麻雀在枝头吵架。

五分钟说走就走。茶凉了，手机响了，又是新的忙忙碌碌。可心里那片装满风、光和人间烟火的小角落，还开着。

窗台上的五分钟，是我和世界之间的一枚小小的逗号。
