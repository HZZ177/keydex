# 文件树筛选、展开、定位与刷新

## 筛选文件

文件树顶部的筛选用于按文件或目录名称缩小范围，不是全文搜索。输入关键词后，会显示名称匹配项及理解目录位置所需的上级路径。

筛选结果有数量和等待时间限制。结果不完整时：

- 使用更具体的名称片段。
- 缩小到明确的目录。
- 清除筛选后手动展开。

要搜索文件内容，应打开文件后使用预览区的内容搜索，或使用项目提供的其他搜索入口。

## 展开目录

- 单击目录展开或收起当前层级。
- 需要展开整个下级结构时，使用目录右键菜单中的“展开所有下级菜单”。
- 大目录可能需要逐步显示内容，等待加载完成后再继续操作。

文件树会按需要加载目录，不必一次展开整个项目。项目很大时，只展开当前任务需要的路径。

## 定位当前文件

文件已经在预览区打开但树中看不到时，使用定位入口，让文件树展开相关上级目录并选中目标文件。若仍没有定位：

1. 确认文件属于当前项目。
2. 清除文件树筛选条件。
3. 刷新目标目录。
4. 外部文件不会被强制显示在项目文件树中。

## 键盘浏览

文件树获得焦点后，可使用方向键移动和展开，Enter 打开当前项。面板内需要快速查找可见文字时，可以尝试 Ctrl+F；这与项目文件内容搜索不是同一功能。

## 刷新

文件或目录被外部程序修改后，可在对应右键菜单中刷新。优先刷新目标文件或目录，不要每次都刷新整个应用。
