# A2UI 选择内容

## 常见形式

选择交互可以显示为卡片画廊或通知式选项列表，并支持：

- 单选：只能选择一个结果。
- 多选：可以组合多个结果。
- 推荐项或默认项。
- 暂不可选的选项。
- 提交后的纠正入口。

## 正确使用

1. 先阅读问题、选项说明和可选数量。
2. 推荐项只是建议，不等于必须选择。
3. 多选时检查组合是否互相冲突。
4. 确认选中状态后再提交。

选项不可用时，查看其说明；可能是前置条件未满足，也可能是该选项只用于展示。不要反复点击禁用项。

## 默认值与推荐值

默认选中表示提交时会采用该值，用户仍应确认是否符合意图。推荐标记说明 Agent 认为它更合适，但最终选择权仍在用户。

涉及删除、覆盖、授权或付费等影响较大的决策时，不能只依赖默认值，必须明确确认。

## 提交后想更改

如果页面提供纠正入口，重新选择并提交；否则在普通消息中说明原选择和新选择，请 Agent 确认是否能继续沿用当前任务结果。

不要假设更改选择会自动撤销已执行的文件或外部操作。
