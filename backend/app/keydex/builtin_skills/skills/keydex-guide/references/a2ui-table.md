# A2UI 可编辑表格

## 表格能做什么

A2UI 表格用于查看和整理多行数据。根据本次内容配置，可能支持：

- 编辑表格标题或单元格。
- 新增行。
- 删除行。
- 点击表头排序查看。
- 提交修改。
- 对已提交内容进行纠正。

不是每张表都开放全部操作，以页面中的可见按钮和可编辑状态为准。

## 编辑建议

1. 先确认列名、单位和每行代表的对象。
2. 编辑时保持同一列的数据格式一致。
3. 新增或删除行后检查总数和关联字段。
4. 提交前复核关键行，尤其是金额、日期和标识信息。

排序通常只改变当前查看顺序，不会自动改写提交数据的业务含义。需要把排序结果作为正式顺序时，应在提交前确认页面是否明确支持。

## 防止误改

大批量编辑前，可以先让 Agent 提供只读预览或说明影响范围。删除行不一定等于删除外部系统中的记录；真正的外部删除仍应经过对应确认或审批。

## 提交后

等待 Agent 根据表格结果继续。页面显示提交成功不等于外部业务操作已经完成，最终应查看后续消息。

需要纠正时使用页面入口；若不可纠正，在普通消息中指出行、列和新值。不要重复提交整张表以尝试覆盖未知状态。
