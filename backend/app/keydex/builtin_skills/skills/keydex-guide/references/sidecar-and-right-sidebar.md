# BTW、Sub-Agent 与右侧栏

## BTW 适合什么

BTW 是不打断主任务的临时旁路对话。适合：

- 询问当前内容中的一个小问题。
- 临时解释术语或比较方案。
- 在不改变主任务方向的情况下补充判断。

可以从 `/btw` 或当前界面提供的 BTW 入口打开。它通常显示在会话右侧栏中。

## 与主会话的区别

- 主会话承载当前正式任务、目标和连续工作。
- BTW 用于临时讨论，不应默认承担需要长期追踪的主任务。
- BTW 中的重要结论若要成为主任务要求，应明确带回主会话。

不要把 BTW 当作新的项目会话，也不要假设它会自动改变主会话正在执行的要求。

## 右侧栏的其他内容

右侧栏还可能显示文件、预览、文件变更审阅和 Sub-Agents。切换这些标签只改变当前查看区域，不会关闭主会话。

“Sub-Agents”列表展示当前任务下的 Explorer 和 Worker，并按活动中、已完成、已取消或失败分组。点击条目可以查看独立子会话；运行中还可发送中途引导或停止。它和 BTW 都位于右侧栏，但 BTW 是用户临时旁路讨论，Sub-Agent 是主 Agent 委派并回收结果的工作会话。

常用操作：

- 拖动边缘调整宽度。
- 使用最大化查看长文或大图。
- 关闭后从对应入口重新打开。

## 内容没有出现时

先确认打开的是 BTW、Sub-Agents 还是文件/预览标签。切换会话再返回时，BTW 会恢复该会话自己的旁路内容；若看起来丢失，先检查当前主会话是否正确，再重新打开 BTW，不要在另一个会话里重复提问。

需要长期保留的结论，应发送回主会话或保存到文件，不要只留在临时旁路讨论中。
