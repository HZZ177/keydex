# 输入框、上下文与附件

## 发送普通消息

在会话底部输入内容并发送。回复进行中再次发送时，消息可能用于引导当前回复或加入队列，具体见“回复进行中时的引导与排队”。

## 添加附件

可使用附件入口选择支持的图片或文件，也可以在界面允许时拖入或粘贴。发送前检查附件是否已经显示在输入框附近；只把文件名写进文字并不等于真正附加了文件。

图片适合用于界面问题、图像分析和视觉参考。普通文件是否可直接读取，取决于当前会话、文件类型和访问范围。

## 引用项目文件

在绑定项目的会话中，输入 `@` 可以查找项目里的文件或目录：

- 选择文件，会把该文件作为上下文引用。
- 选中目录后按 Enter，可继续浏览目录内容。
- 需要直接引用整个目录时，使用界面提示的组合键；当前通常为 Ctrl+Enter。

在文件树或预览区使用“添加到聊天/会话”也能完成引用。

普通 Chat 没有项目工作区，因此不会提供项目文件引用。若用户需要 `@` 项目文件，应新建或切换到绑定项目的会话。

## 斜杠入口

输入 `/` 可查看当前支持的快捷入口，例如：

- `/btw`：打开临时旁路对话。
- `/goal`：处理会话目标。
- `/compact`：整理较长会话的上下文。
- `/skill`：选择或使用 Skill。

以当前弹出的列表为准；不要向用户承诺列表中不存在的命令。

## 发送前检查

当任务依赖特定文件或图片时，确认引用标签或附件预览仍在输入框中。切换会话后，应重新检查附件是否属于当前会话。
