# 大纲、图片、图表与富内容预览

## Markdown 大纲

带标题结构的 Markdown 文档可以显示大纲。点击大纲项可跳转到对应标题，适合浏览长文。

大纲缺少某一项时，先检查源码中的标题格式是否正确，再刷新预览。普通加粗文字不会自动成为标题。

## 图片预览

图片预览常用操作包括：

- 放大和缩小。
- 旋转。
- 重置显示。
- 全屏查看。

放大或旋转只影响查看方式，不会改写原图片文件。需要实际裁剪、改色或另存时，应使用明确的图像处理功能。

## 代码块

长代码块可通过当前提供的入口放大查看，或在侧边区域展开。复制前确认是否需要包含语言标记、行号或只复制代码正文。

## Mermaid 图表

Mermaid 内容会以图表方式预览，并提供适合查看的缩放或重新定位操作。图表无法渲染时：

1. 切换源码查看语法。
2. 修正不完整的节点、连线或代码围栏。
3. 返回预览重新确认。

渲染失败不等于源文件为空。

## HTML 与其他富内容

项目内 `.html` 文件可通过受控本地地址加载同一工作区中的相对脚本、样式和图片；消息中的 HTML 代码块也可展开或全屏查看。使用 Vite 等本机开发页时需要先启动对应 loopback 服务。

脚本仍在隔离的预览环境中执行，项目外资源、远程内容和浏览器安全策略可能限制部分效果。需要判断文件是否正确时，同时核对源码、资源路径和开发服务；不要把预览限制当成源文件缺失。
