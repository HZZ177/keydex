# 大纲、图片、图表与富内容预览

## Markdown 大纲

带标题结构的 Markdown 文档可以显示大纲。点击大纲项可跳转到对应标题，适合浏览长文。

大纲缺少某一项时，先检查源码中的标题格式是否正确，再刷新预览。普通加粗文字不会自动成为标题。

## 图片预览

图片预览常用操作包括：

- 放大和缩小。
- 旋转。
- 重置显示。
- 全屏查看。

放大或旋转只影响查看方式，不会改写原图片文件。需要实际裁剪、改色或另存时，应使用明确的图像处理功能。

## 代码块

长代码块可通过当前提供的入口放大查看，或在侧边区域展开。复制前确认是否需要包含语言标记、行号或只复制代码正文。

## Mermaid 图表

Mermaid 内容会以图表方式预览，并提供适合查看的缩放或重新定位操作。图表无法渲染时：

1. 切换源码查看语法。
2. 修正不完整的节点、连线或代码围栏。
3. 返回预览重新确认。

渲染失败不等于源文件为空。

## HTML 与其他富内容

出于安全和兼容性考虑，部分脚本、外部资源或交互效果可能不会在预览中执行。需要判断文件是否正确时，同时核对源码；不要把预览限制当成源文件缺失。
