# A2UI 图表

## 适合展示什么

图表用于把数据关系直接展示在会话中。当前常见样式包括：

- 趋势图：查看数据随时间或顺序变化。
- 柱状图：比较不同项目或分类。
- 饼图：查看组成比例。
- 桑基图：查看多个环节之间的流向和分布。

## 阅读图表

先看标题、单位、时间范围和图例，再比较数据。鼠标悬停等操作可能显示更具体的数值，以当前页面提供的交互为准。

图表显示的是 Agent 本次提供的数据，不代表它会自动跟随外部数据源更新。需要最新数据时，应让 Agent 重新查询或生成。

## 当前边界

A2UI 图表主要用于查看，不应指导用户直接在图表中修改原始数据。需要筛选、编辑或提交数据时，使用选择、表单或表格交互。

## 显示异常

遇到空白、标签重叠或数据难以辨认时：

1. 扩大会话或右侧面板的可用空间。
2. 查看消息中是否有数据范围说明。
3. 让 Agent 改用更合适的图表或分组。
4. 必要时要求同时给出文字摘要或表格。

不要仅凭图形面积估算精确数值；需要准确数字时读取提示或请求表格版。
