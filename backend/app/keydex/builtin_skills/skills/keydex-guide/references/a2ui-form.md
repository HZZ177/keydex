# A2UI 表单

## 支持的常见字段

A2UI 表单可包含：

- 单行文本和多行文本。
- 数字。
- 开关或布尔选择。
- 单选下拉和多选下拉。
- 日期。

字段旁的说明、占位内容、必填标记和错误提示用于帮助正确填写。

## 填写步骤

1. 先浏览全部字段，确认需要准备的信息。
2. 按字段要求输入，注意数字、日期和多选格式。
3. 处理页面显示的校验提示。
4. 提交前再次检查敏感信息和高影响选择。

提交按钮不可用时，通常需要补齐必填项或修正格式。滚动查看表单中是否还有未处理的提示。

## 敏感信息

除非任务确实需要且当前会话范围可信，不要在普通表单中输入密码、密钥或完整个人隐私信息。正式的登录授权应使用对应服务提供的认证入口。

## 提交与取消

提交后等待 Agent 确认，不要因短暂等待重复提交。取消本次表单后，可以通过普通消息解释原因并请求其他处理方式。

## 填错后的处理

页面提供纠正入口时直接修改；没有时，在普通消息中指出字段名、原值和希望改成的值。涉及已经执行的外部操作时，同时确认是否需要补救，而不是只修改表单显示。
