# 文档批注与发送到会话

## 两类批注

- **选中文本批注**：针对文档中的具体句子或段落。
- **文档级批注**：针对整份文档，不依赖某一段选区。

选中文本后，可以只把原文引用到会话，也可以写下意见并保存为可持续管理的批注。临时引用适合一次性提问；持久批注适合审阅、修改清单和后续回看。

## 创建批注

1. 打开支持批注的文档。
2. 选中目标文本，或从文档菜单添加文档级批注。
3. 输入明确的意见、问题或修改要求。
4. 保存后在批注栏中查看。

一条批注尽量只表达一个问题，并说明期望结果。这样更适合逐项发送给 Agent 处理。

## 管理与导航

批注栏支持在批注之间跳转，并根据当前界面提供编辑、删除和添加到会话等操作。可以发送单条批注，也可以把多条批注一起加入会话。

批量发送前检查批注之间是否冲突。需要 Agent 按顺序处理时，在消息中说明优先级或验收方式。

## 文本变化后的重新定位

文档内容修改后，原批注选区可能无法唯一对应。出现位置不明确、文本已变化或无法跳转时：

1. 打开当前版本文档。
2. 找到正确的新文本。
3. 重新选择并关联批注。
4. 再添加到会话。

不要让 Agent 根据已经错位的旧选区直接修改。

## 可用范围

文档未保存、已从磁盘删除、来自只读 Skill 资源，或是当前项目外的受限文件时，批注入口可能不可用。文档级批注需要文档已经正常打开和解析。

把批注加入会话时，Keydex 会尽量带入当前可解析的内容。发送前仍应核对引用文本是否正确。
