# 文件和目录右键操作

## 目录右键菜单

在项目文件树中右键目录，当前可用操作包括：

- **展开所有下级菜单**：展开该目录下的完整可见层级。
- **添加该目录到会话**：把目录引用带入当前会话的输入区。
- **刷新**：重新读取该目录的当前内容。

添加目录后应检查输入区中的引用，再发送消息。展开所有下级目录可能显示大量项目项，较大项目中按需使用。

## 文件右键菜单

右键文件时，当前可用操作包括：

- **复制文件**：将文件作为可粘贴的文件对象复制。
- **打开于 → 资源管理器**：在 Windows 资源管理器中定位或打开。
- **复制绝对路径**：复制完整磁盘路径。
- **复制工作区相对路径**：复制相对于当前项目的位置。
- **添加到聊天**：把文件引用加入当前会话输入区。
- **刷新**：重新读取该文件。

绝对路径适合本机定位；工作区相对路径更适合项目内讨论和分享。发送给其他人前，应避免暴露不必要的本机用户名或目录结构。

## 预览内容中的右键菜单

在预览或源码编辑区域中，右键菜单会根据当前目标提供剪切、复制、粘贴、全选，以及添加文件到会话、添加文档批注、刷新等操作。只读内容不会提供可修改内容的操作。

## 当前不应声称存在的操作

文件树右键菜单当前不提供新建、重命名、移动或删除文件/目录的入口。用户需要这些操作时，可以让 Agent 在获得相应权限后处理，或使用系统文件管理工具；不要指导用户去寻找不存在的菜单项。
