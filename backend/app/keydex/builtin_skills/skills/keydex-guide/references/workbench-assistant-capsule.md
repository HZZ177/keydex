# 工作台助手与胶囊面板

## 助手的几种显示方式

工作台助手会根据当前任务和空间，以不同大小出现：

- **胶囊**：占用空间最小，显示助手入口或当前状态。
- **快捷输入**：适合直接提出简短要求。
- **展开面板**：在不完全遮挡文件的情况下查看更多会话内容。
- **抽屉**：适合处理较长对话、审批、A2UI 或完整消息记录。

这些是同一个工作台助手的不同显示方式，不代表创建了多个会话。

## 选择合适的方式

- 主要在读文件，只偶尔提问：保持胶囊或快捷输入。
- 需要边看文件边持续对话：使用展开面板。
- 需要集中处理长回复或交互内容：打开抽屉。

展开或收起助手不会删除对话。关闭抽屉后，可从胶囊重新打开。

## 与当前文件协作

需要 Agent 处理当前文件时，明确引用文件或使用预览区的“添加到会话”入口。仅仅把文件显示在主预览中，不一定等于已经把它加入本轮消息。

批注、选中文本和文件引用可以带入助手。发送前检查输入区中是否出现对应引用。

## 状态提示

胶囊可能显示正在回复、等待审批、等待输入或已完成等状态。遇到等待状态时，打开助手查看具体请求；只反复点击胶囊不会自动完成审批或表单。

## 位置与尺寸问题

助手遮挡文件时，优先收起或切换为较小形态。需要同时查看长文和对话时，调整工作台分隔线或打开抽屉，不要把全局右侧栏误当作工作台助手。
