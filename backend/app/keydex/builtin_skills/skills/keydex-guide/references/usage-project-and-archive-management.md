# 用量、项目与归档管理

## 用量页面

进入“设置 → 用量统计”查看模型或会话的使用情况。可以根据当前提供的筛选项按时间、服务、模型或会话缩小范围，查看趋势、明细并在界面允许时导出。

需要追查某次异常消耗时，先缩小时间范围，再从详情跳转到对应会话。用量记录反映实际请求情况，不应只根据消息数量估算。

不同模型服务的计量和费用规则可能不同。Keydex 显示的用量不一定等同于服务商最终账单，涉及结算时以服务商账单为准。

## 项目管理

进入“设置 → 项目管理”查看已经添加的项目位置，并处理失效或不再使用的项目记录。移除项目登记不会自动删除磁盘上的项目目录。

项目被移动后，旧入口可能无法打开。移除旧记录，再从新位置添加。执行前确认是否还有依赖旧路径的会话或引用。

## 归档管理

进入“设置 → 归档管理”查看已收起的会话，并执行恢复或永久删除。恢复后会话重新出现在正常列表中。

永久删除前：

- 确认是否需要导出 Markdown。
- 检查是否包含仍需回看的审批、批注或任务结果。
- 区分删除会话和删除项目文件，两者不是同一操作。

## 找不到记录

先清除筛选条件，再确认当前时间范围、项目分组和归档状态。用量记录与会话列表使用不同入口，不要在一个页面中寻找另一个页面的全部内容。
