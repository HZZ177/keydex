# 预览格式、源码与分屏模式

## 支持的常见内容

工作台或右侧预览可根据文件类型显示不同内容，包括：

- Markdown、HTML 与纯文本。
- JSON 和常见代码文件。
- Diff 差异内容。
- Mermaid 图表。
- 图片。

不支持的格式可能只提供基础文本查看、外部打开，或提示无法预览。

## Markdown 与 HTML

对适合渲染的文件，可在以下模式间切换：

- **预览**：查看排版后的内容。
- **源码**：查看并编辑原始文本。
- **分屏**：同时查看源码与预览，适合边写边核对效果。

不同文件或只读资源可能只开放其中一部分模式。没有源码入口时，先确认文件是否只读或来自项目外部。

## 内容搜索与复制

需要查找正文时，使用预览区的搜索入口，而不是文件树名称筛选。搜索结果可逐项跳转。

复制时注意当前所在模式：预览模式复制的是渲染后的可见内容，源码模式复制的是原始文本。需要保留 Markdown 标记时使用源码模式。

## 关闭预览

关闭标签或预览面板不会删除文件。再次从文件树或消息入口打开即可。

## 显示异常

如果预览内容与源码不一致：

1. 检查源码是否仍有未保存或冲突提示。
2. 切换到源码确认真实文本。
3. 刷新该文件。
4. 对 HTML 等内容，确认当前安全限制是否阻止了部分外部内容。

不要仅凭渲染效果判断文件已经丢失或被改写。
