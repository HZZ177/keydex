# 扩展功能设置

## 扩展页面负责什么

扩展设置用于管理会话中的增强能力。当前可能包括：

- 自动生成会话标题。
- 防止 Agent 无意义重复操作的保护。
- 长会话的上下文整理。
- A2UI 展示和调试。
- Web 搜索与页面读取。

不同版本提供的项目可能不同，以当前设置页面为准。

## 会话标题

启用后，Keydex 会根据会话内容生成便于识别的标题。用户仍可在会话菜单中手动重命名。关闭自动标题不会删除已有标题。

## 重复操作保护

当 Agent 反复执行相似操作却没有进展时，保护机制可提醒或阻止继续。触发后应检查任务是否缺少信息、权限或正确路径，不要立即关闭保护继续重试。

## 上下文整理

较长会话可启用或手动触发上下文整理。调整相关设置后，用重要限制明确的长任务验证，确保目标和未完成事项仍被保留。

## A2UI

普通用户只需保持 A2UI 展示开启。调试入口用于排查交互内容，不需要日常开启。关闭 A2UI 后，依赖表单、选择或表格的任务可能无法按原方式继续。

## Web

启用 Web 后还需要配置可用服务和凭据。仅打开总开关但没有完成服务测试时，Agent 仍可能无法联网。

调整扩展设置通常影响之后的任务；已经生成的消息不会被重新处理。
